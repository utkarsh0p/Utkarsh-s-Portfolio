# Utkarsh's Portfolio

A macOS-inspired portfolio website focused on a clean user interface, smooth animations, and a modern desktop-like experience.

## Tech Stack

- HTML  
- Tailwind CSS  
- React.js  
- Zustand for state management  
- GSAP and Draggable for animations and window interactions  

## Features

- macOS-style UI design  
- Draggable windows  
- Smooth GSAP animations  
- Custom dock with interactive icons  
- Responsive layout for all screen sizes  
- Simple and organized React component structure

npm run dev
